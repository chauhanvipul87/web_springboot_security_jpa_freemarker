<h2>Spring Boot + Spring Security+ JPA -Mysql + Freemarker + Jboss - Rest Easy project </h2>

 1) JPA integration with MYSQL <br/>
 2) Spring security validate agains the MYSQL user table<br/>

URL-based security <br/>
